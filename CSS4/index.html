"Position - Static , Relative, Absolute, Fixed, Sticky"

  • Position relative is always related to the root.
  • Position absolute is related to parent if it doesnot find the parent then it will go for root.
  • Position Static is having by default thing, we cant give any power like left....
  • Position:fixed it takes fixed position, its not move , if we scroll the page.
  • Position:sticky , combination of relative and fixed ,position takes wrt parent.
